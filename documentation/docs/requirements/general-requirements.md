# General Requirements
1. The system shall support extracting context from "Slack" channel conversations.
2. The system shall exclude direct messages and private channels by default unless otherwise enabled.
3. The system shall support a user-controlled extraction model (opt-in or hybrid approval).
4. The system shall store structured context in a persistent knowledge store.
5. The system shall provide a user-facing method to retrieve context (chatbot interface, dashboard, or both).
6. The system shall be modular enough to allow future support for other platforms like Discord.
